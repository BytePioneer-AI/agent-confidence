# Word 经营分析报告 Agent 可信等级设计审查

## 1. 结论

- 设计成熟度：可试点
- 数字和结构使用确定性检查；原因分析同时要求证据链接和语义风险审查
- 语义审查“未发现风险”不是独立高可信证据，高可信还依赖来源、期间、数字和证据映射全部通过

## 2. 关键控制

- 事实指标不一致直接判低；
- 必要章节缺失判低；
- 因果声明缺少证据或语义审查不可用时，原因分析最高为中；
- 因果声明发现缺少依据时，标记具体段落并要求定向核查。

## 3. 人工审查动作

- 指标冲突：核对 metric_id、期间、单位和来源引用；
- 缺少证据：检查 claims 中的 evidence_refs；
- 因果风险：核对是否存在支持原因的独立数据，或改写为可能性表述。
