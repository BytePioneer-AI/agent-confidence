# 示例索引

## Excel 销售汇总

参考仓库：

```text
examples/excel-sales-report/confidence-design.md
```

重点：

- 拆分明细、汇总、报告指标和输出文件；
- 把混合日期、隐藏行遗漏、跨文件数字不一致纳入等级；
- 建议将逻辑分别融入导入流程、清洗代码、SQL 汇总和报告生成后检查。

## Word 经营报告

参考仓库：

```text
examples/word-management-report/confidence-design.md
```

重点：

- 区分报告结构、事实指标、趋势描述、原因分析和行动建议；
- 对事实错误和语义推断采用不同等级条件；
- 将数字核对融入数据和报告流程，将因果风险放入语义审查或人工审核。

## 使用示例时

不要照搬具体条件。先根据当前业务确认：

- 结果单元；
- 关键性；
- 验收标准；
- 历史问题；
- 现有实现位置；
- 下游风险。
